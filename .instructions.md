## WINRATE BOOST — SCORING ENGINE

OBJECTIF :
Filtrer les trades faibles et n’autoriser que les entrées à forte probabilité.

---

## SCORE GLOBAL (0 → 100)

Le score est calculé AVANT toute entrée.

COMPOSANTS :

1. STRUCTURE (0-25)
- tendance claire = +25
- range = +10
- chaos = 0

2. MULTI TIMEFRAME (0-25)
- M1 + M5 + M15 alignés = +25
- partiellement alignés = +10
- divergents = 0

3. LIQUIDITÉ / ZONES (0-20)
- zone propre (support/résistance / FVG / OB) = +20
- zone moyenne = +10
- aucune zone = 0

4. IMPULSION / VOLUME (0-15)
- impulsion forte = +15
- moyenne = +8
- faible = 0

5. TIMING (0-15)
- pullback propre = +15
- entrée tardive = +5
- mauvais timing = 0

---

## DÉCISION

- SCORE ≥ 75 → ENTRÉE AUTORISÉE
- SCORE 60–74 → ATTENTE / CONFIRMATION
- SCORE < 60 → INTERDIT

INTERDICTION :
- entrer sous 60
- forcer un trade

---
